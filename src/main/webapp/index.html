Eulerity Take-Home Challenge

Submit a URL to crawl and discover images & logos

How it works

  • Submit the form to POST a URL to the backend.
  • The backend will crawl and extract image URLs.
  • Logos are highlighted distinctly.

Developer Notes

  • Extended client-side URL validation using the native URL constructor to prevent malformed requests.
  • Added immediate UX feedback for invalid or empty input before any API call is triggered.
  • Introduced protocol checks to ensure only http and https URLs are accepted.
  • Preserved modular design by keeping validation logic isolated from API response handling.
  • Minimized unnecessary network requests, reducing server load and improving performance.
  • Maintained consistent error display styles for better visual feedback and user experience.
